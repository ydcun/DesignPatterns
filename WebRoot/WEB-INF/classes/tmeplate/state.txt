场景：
	两个学生做回答试卷题目，题目都是相同的；只有答案是不同的

分析：
	1.根据面向对象设计思想，继承的优点来将相同的题目放到试卷类中，让每个学生来继承他从而起到公用题目的目的
	2.为了让答题规范，在父类中通过抽象的方法类规范每道题目的答题方法，让子类去实现
	
步骤：
	1.创建TestPaper类其中有三个打印题目的方法，同时有三个回答题目的抽象方法
	2.创建两个学生类StudentA，StudentB 让这两个类集成TestPaPer并重写回答问题方法
	3.创建Main函数来进行测试
	public static void main(String[] args) {
		TestPaper paperA = new StudentA();
		TestPaper paperB = new StudentB();
		

		paperA.answer1();
		paperA.answer2();
		paperA.answer3();
		
		paperB.answer1();
		paperB.answer2();
		paperB.answer3();
	}

总结：
	当不变和可变的行为在方法的子类实现中混合出现的时候，
	不变的行为就会在子类中重复出现。
	我们通过模板方法模式吧这些行为搬移到单一的地方，
	这样就帮助子类摆脱重复不变的行为纠缠
